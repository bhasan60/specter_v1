## Bill of Materials (BOM)
* 1x Seeed Studio XIAO RP2040
* 9x Mechanical Switches
* 9x 1N4148 Diodes
* 1x Rotary Encoder (with push-button)
* 1x OLED Display
* 3D Printed Case (Top and Bottom)
